How to host a node.js project for free without using Vercel, Render, or any third party hosting provider


After searching long on internet, I finally found a way to host node.js file on replit for free. Without any third party hosting provider like Vercel, Render, Deta etc.


Go to your node.js project and enter this block of code.

Create a folder named public and create main.html file inside

Get your project link from replit

In order to get your project link from replit. Enter your project name then your replit username and then repl.co
https://{project name}.{replit username}.repl.co


Open cron-job.org and signup


Signup on cron-job and create a cron job by using your project link. You can set up time for cron job, usually 5 min is good option for it.
